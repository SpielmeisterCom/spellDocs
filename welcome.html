SpellJS

The SpellJS framework enables you to develop games for the web and mobile platforms. It hides the additional effort associated with the development of cross-platform applications so that you can concentrate on creating games instead of building cross-platform infrastructure.

The integrated development enviroment SpellEd assists with the creation of content.
